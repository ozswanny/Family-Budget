# Rodney & Pearl — Family Budget Planner

A personal household budget planner for Rodney and Pearl.

## Features
- 12-month budget planning with income & expenditure tracking
- Australian Dollar (A$) currency
- Dual income streams (Rodney & Pearl) with Pension, Teaching, Art categories
- Recurring items that populate all months automatically
- Year-to-date cumulative totals per row and per category
- Multi-year support (add years as needed)
- Colour coded: Green = Budgeted, Blue = Actual, Red = Deficit
- Copy month forward, Save/Load backup, CSV export
- Data saves automatically in your browser

## Using the app
Open the link and start entering your budget. Your data saves automatically
to your browser's local storage — use the **Save Backup** button regularly
to download a `.json` backup file for safekeeping.

## Restoring data
Click **Load Backup** and select a previously saved `.json` file to restore
all your data on any device.
